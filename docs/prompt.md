# 大纲例子
1. **Reward Modeling**
   - **奖励的作用**：奖励是训练信号的来源，决定了强化学习（RL）的优化方向。
   - **规则奖励系统**：
     1. **准确性奖励（Accuracy rewards）**：评估模型响应是否正确。例如，在数学问题中，要求模型以特定格式（如框内）提供最终答案，便于基于规则的验证。对于LeetCode问题，可以使用编译器根据预定义的测试用例生成反馈。
     2. **格式奖励（Format rewards）**：要求模型在`<think>`和`</think>`标签之间展示其思考过程。
   - **未使用神经奖励模型的原因**：在大规模强化学习过程中，神经奖励模型可能面临奖励欺骗（reward hacking）问题，且重新训练奖励模型需要额外的资源，增加了训练流程的复杂性。
2. **Training Template**
   - **模板设计**：设计了一个简单的模板，指导基础模型遵循指定的指令。
   - **模板结构**：要求模型首先生成推理过程，然后给出最终答案。
   - **设计原则**：避免内容特定的偏见（如强制反思性推理或特定的问题解决策略），以确保在RL过程中准确观察模型的自然进展。
